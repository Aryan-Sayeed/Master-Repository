3. “Dog Barks”
⭐ What you’ll learn:
Using a class with simple behavior (methods)

✅ Example idea:
Dog class with:

name (string)

breed (string)

Method: bark()

Print: "Woof! My name is Max and I’m a Labrador!"

📁 Why this works:
It feels like you’re giving an object real “behavior” — the point of OOP.
